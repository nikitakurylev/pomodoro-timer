# pomodoro-timer

A simple [pomodoro](https://en.wikipedia.org/wiki/Pomodoro_Technique) timer for VS Code

## Features

`Start Timer` - starts timer

`Stop Timer` - stops timer

`Reset Timer` - stops timer and resets it back to 25 minutes
